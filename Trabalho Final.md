# Trabalho Final — Front End

## Objetivo

O objetivo deste trabalho é reproduzir o site abaixo utilizando apenas tecnologias de Front End:

[Wanderland](https://wanderland.qodeinteractive.com/)

É obrigatório reproduzir tudo até a seção "Featured Blog Posts", incluindo também o footer do site. 

---

# Requisitos Obrigatórios

O projeto deve conter:

- Estrutura HTML semântica
- Estilização completa com CSS
- Responsividade
- Animações presentes no site original
- Layout semelhante ao original
- Navbar funcional
- Seções organizadas corretamente
- Footer completo
- Utilização de unidades flexíveis quando necessário (`%`, `vh`, `vw`, `rem`, `em`, `flex`, etc.)

---

# Responsividade

O site deve funcionar corretamente em:

- Desktop
- Tablet
- Celular

Utilize:

- Media Queries
- Flexbox
- Unidades responsivas

---

# Animações

As animações do site original devem ser reproduzidas o mais fielmente possível.

Exemplos:

- Hover effects
- Transições suaves
- Animações ao rolar a página
- Efeitos de entrada
- Transformações com CSS

Pode ser utilizado JavaScript para auxiliar nas animações.

---

# Imagens

As imagens devem ser obtidas através do navegador utilizando o recurso:

- Inspecionar Elemento

Não é permitido substituir por imagens aleatórias.

---

# Tecnologias Permitidas

- HTML5
- CSS3
- JavaScript

Bibliotecas são permitidas apenas para animações, caso necessário.

---

# Organização do Projeto

O projeto deve possuir organização adequada de pastas.

Exemplo:

```txt
/assets
    /images
    /styles
    /scripts

index.html
````

---

# Critérios de Avaliação

Serão avaliados:

* Semelhança visual com o site original
* Responsividade
* Organização do código
* Qualidade do CSS
* Fidelidade das animações
* Estrutura HTML
* Capricho visual
* Funcionamento geral
